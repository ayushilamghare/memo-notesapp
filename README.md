Notes App
A simple and responsive notes management app built using React. Users can create, organize, and manage notes with features like tagging, search, pinning, archiving, and trash.

Features: 
Create, edit, and delete notes
Search notes by title or content
Add and filter notes using tags
Pin important notes
Archive notes
Move notes to trash and restore or delete permanently
Data stored using localStorage

Tech Stack:
React JS
Tailwind CSS
React Router DOM
localStorage
